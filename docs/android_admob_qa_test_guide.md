# AdMob 보상형 광고 QA 테스트

이 빌드는 실제 AlphaMate AdMob 앱 ID와 보상형 광고 단위 ID를 사용해 `광고 시청 -> 서버 검증 -> 심화 복기 이용권 지급`을 확인하는 전용 APK입니다. 배너와 전면 광고는 아직 실제 광고 단위가 없으므로 Google 데모 ID를 유지합니다.

## APK 만들기

프로젝트 폴더에서 `verify_android_admob_qa.bat`를 더블클릭합니다.

완성된 파일:

```text
D:\Project\Vibe\StockAnalyze\frontend\android\app\build\outputs\apk\qa\alphamate-admob-qa.apk
```

## 테스트 전 주의사항

- AdMob에 등록한 테스트 기기에서만 설치하고 실행합니다.
- 광고 화면에 `테스트 광고` 또는 `Test Ad` 표시가 있는지 확인합니다.
- 표시가 없다면 광고를 누르거나 끝까지 시청하지 말고, 테스트 기기 등록이 반영될 때까지 기다립니다. 반영에는 시간이 걸릴 수 있습니다.
- 앱의 API Key나 서버 비밀값은 APK에 포함되지 않습니다.

## 심화 복기 이용권 확인

1. 앱에서 카카오 또는 네이버 로그인을 완료합니다.
2. `매매복기` 화면의 이용권 영역으로 이동합니다.
3. 심화 복기 이용권 광고 버튼을 누르고 보상형 광고를 끝까지 시청합니다.
4. 현재 정책상 광고 5회 시청을 완료하면 주간 심화 복기 이용권 1장이 지급됩니다.
5. 이용권 숫자가 1로 바뀌는지 확인한 뒤 심화 복기를 실행합니다.

실패하면 오류 팝업 화면과 발생 시각을 함께 기록합니다. 광고 ID, API Key, 로그인 비밀값은 캡처하거나 전달하지 않습니다.
