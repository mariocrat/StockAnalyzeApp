# 광고 노출 정책

- 최초 실행은 광고 없이 Pulse 스플래시와 초기 테마 데이터 로딩만 사용한다.
- 앱이 백그라운드로 내려간 뒤 90초 이상 지나 다시 foreground로 돌아오면 App Open 광고를 1회 시도한다.
- 앱 복귀 App Open 광고는 같은 세션에서 10분 쿨다운을 둔다.
- 최초 실행에서는 광고를 띄우지 않고 App Open 광고를 미리 로드하며, 로드 후 4시간이 지난 광고는 다시 불러온다.
- 차트 `자세히 보기` 진입은 무료 사용자 기준 3번째 진입마다 전면 광고를 시도한다.
- 광고 로드 실패, 앱 실행, SDK 미사용 환경에서는 사용자 흐름과 차트 상세 진입을 막지 않는다.
- Pro 구독자는 앱 전체에서 보상형 광고를 제외한 비보상 광고를 보지 않도록 전역 광고 정책에서 차단한다.
- 네이티브 광고는 테마/종목 리스트의 선택 흐름과 탐색 속도에 영향을 줄 수 있어 이번 단계에서는 실제 삽입하지 않고 정책으로만 남긴다.
